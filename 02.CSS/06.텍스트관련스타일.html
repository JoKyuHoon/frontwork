3. text-transform : 영문 텍스트의 대소문자 변환

        선택자 {
            text-transform : uppercase|lowercase|capitalize;
        }
  1. uppercase : 모든 영문자를 대문자로
  2. LOWERCASE : 모든 영문자를 소문자로
  3. capitalize : 첫 글자를 대문자로 first text

4. text-shadow : 텍스트에 그림자 효과

        선택자 {
            text-shadow : 가로거리(x축), 세로거리(y축) [번짐정도] [색상];
        }
